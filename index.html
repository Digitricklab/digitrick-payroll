Deploy Apps Script JSON API

Extensions → Apps Script → paste code → Deploy as Web App (Anyone access):

function doGet(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); const tab = e.parameter.sheet || 'Employees'; const [headers, ...rows] = ss.getSheetByName(tab).getDataRange().getValues(); return ContentService.createTextOutput(JSON.stringify( {data: rows.map(r => Object.fromEntries(headers.map((h,i)=>[h,r[i]])))} )).setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const p = JSON.parse(e.postData.contents); SpreadsheetApp.getActiveSpreadsheet().getSheetByName(p.sheet).appendRow(p.row); return ContentService.createTextOutput('{"status":"ok"}').setMimeType(ContentService.MimeType.JSON); }

GitHub Repository

Monthly Leave Auto-Credit (Apps Script Trigger)

Add this function and set a Time-driven trigger: Month timer → 1st of month:

function addMonthlyLeave() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('LeaveBalance'); const data = sheet.getDataRange().getValues(); for (let i = 1; i < data.length; i++) { // Column C = balance, add 1 leave credit per month sheet.getRange(i+1, 3).setValue(Number(data[i][2]) + 1); } // Trigger: Edit → Current project's triggers → Add trigger → addMonthlyLeave → Time-driven → Month timer }

Attendance Policy Rules

Digitrick Lab shift and deduction policy.

⏰ Shift Timing

10:00 AM – 6:30 PM
Net working: 7.5 hrs/day
1 hr lunch break excluded

✅ Grace Period

10 min grace — login by 10:10 AM = On Time. No deduction.

⚡ Extra Grace (3×/month)

Extra 5 min — up to 10:15 AM. Max 3 uses. Flagged, no pay cut.

🔴 Late Arrival

After 10:15 (or quota used) = Late. Minutes from 10:00 deducted at per-minute rate.

📋 Paid Leave

Approved leave = full salary. Zero LOP. Requires leave balance (except LOP type).

❌ LOP (Absent)

Absent = Loss of Pay. (CTC ÷ 30) × absent days deducted automatically.

🕐 Permission Policy

120 min/month total. 1 hr × 2 times or 2 hrs × 1 time. Excess deducted.

🌿 Leave Balance

1 leave auto-credited after each completed month. Accrues over time. Apply via Leave tab.

Deduction Formula
Daily Rate    = CTC ÷ 30     Hourly = Daily ÷ 7.5     Per-Min = Hourly ÷ 60
──────────────────────────────────────────────
LOP         = Absent Days × Daily Rate
Late        = Late Mins × Per-Min Rate
Perm Over   = (Perm Used − 120) × Per-Min Rate
Short Hrs   = (7.5 − Net Hrs) × 60 × Per-Min Rate
Net Pay     = Gross − LOP − Late − Perm Over − Short Hrs
